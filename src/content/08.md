# Configuration

- Command line parameters for `tsc`
    - `--noImplicitAny`
- `tsconfig.json`
    - ` ..., "noImplicitAny": "true", ...`

--

## Interesting Options

- no implicit any 
- strict null checks
- no unused locals
- no unused params
- target (es2015, es5)
- library (es2016, node, ...)
- React: jsx, jsx factory
- Angular: experimental decorators
